### 万字文档+4S店客户管理系统+Uniapp+SSM+Vue+sql文件+软件环境安装包
## 项目采用技术：
开发环境：IDEA、JDK1.8、Maven、MySQL、Tomcat7.5-9、微信开发者工具/HBuilderX
技术栈：Java、MySQL、Tomcat7.5-9、Spring、SpringMVC、Mybatis/MybatisPlus、Vue2、Uniapp、ElementUI、HTML5（H5）、JavaScript、CSS、图片上传
采用M（model）V（view）C（controller）三层体系结构，MySQL数据库作为系统数据储存平台，实现了基于B/S结构的Web系统加小程序。

系统功能模块：
1.管理员服务端；首页、个人中心、用户管理、门店管理、车展管理、汽车品牌管理、新闻头条管理、预约试驾管理、我的收藏管理、系统管理。
2.用户客户端：首页、车展、新闻头条、我的。
